# MCP Selection Sheet

## Project

**Smart Task Escalation Engine**

## Objective

The Smart Task Escalation Engine uses Model Context Protocol (MCP) servers to provide AI coding agents with real-time access to framework documentation, database references, testing guidance, and cloud service standards.

---

## MCP Selection Matrix

| MCP Server | Technology | Purpose | Benefit |
| ---------- | ---------- | ------- | ------- |
| Next.js MCP | Next.js | Routing, SSR, API routes, middleware | Framework-compliant frontend code |
| React MCP | React | Components, hooks, state management | Reusable UI components |
| Node.js MCP | Node.js | Runtime, async patterns, backend standards | API and service-layer generation |
| PostgreSQL MCP | PostgreSQL | Schema design, queries, indexing | Optimized schemas and migrations |
| Playwright MCP | Playwright | Browser automation, assertions | End-to-end test generation |
| AWS MCP | AWS | SES, Secrets Manager, IAM | Cloud integration guidance |

---

## MCP Usage by Layer

| Layer | MCP Servers | Usage |
| ----- | ----------- | ----- |
| Frontend | Next.js, React | Login, dashboard, escalation UI, API client |
| Backend | Node.js, PostgreSQL | REST APIs, escalation logic, auth, audit |
| Testing | Playwright | E2E escalation workflow validation |
| Cloud | AWS | SES notifications, secrets, deployment |
